# Clients - User Guide

## What is a Client?
A client is a qualified person who we're actively helping get a mortgage. They were either converted from a lead or added directly.

---

## What Can I Do?

### View Clients
- **As a user, I can** see all my clients in a table
- **As a user, I can** switch between "Active" tab and "All" tab
- **As a user, I can** search clients by name, phone, or email
- **As a user, I can** see their eligibility status (Eligible, Not Eligible, Pending)
- **As a user, I can** see their active cases

### Create New Client
- **As a user, I can** click the "+" button to add a new client
- **As a user, I can** enter their contact details
- **As a user, I can** enter their financial details (salary, loan amount, property value)

### View Client Details
- **As a user, I can** click on any client to open their details panel
- **As a user, I can** see their contact info
- **As a user, I can** see their financial info (salary, employer, loan details)
- **As a user, I can** see calculated DBR (Debt Burden Ratio) and LTV (Loan to Value)
- **As a user, I can** see their eligibility status

### Edit Client Info
- **As a user, I can** click the pencil icon to edit client details
- **As a user, I can** update contact info, financial info, or employment details
- **As a user, I can** save or cancel my changes
- **As a user, I can** see eligibility recalculated after saving

### Manage Documents
- **As a user, I can** click the "Documents" tab to see all documents
- **As a user, I can** see which documents are uploaded, missing, or pending
- **As a user, I can** upload new documents (Emirates ID, Salary Certificate, Bank Statements, etc.)
- **As a user, I can** delete uploaded documents
- **As a user, I can** upload additional "Other" documents

### Log Calls
- **As a user, I can** click the phone icon to log a call
- **As a user, I can** select the call outcome and add notes

### Add Notes
- **As a user, I can** click the note icon to add a note
- **As a user, I can** write any important information

### Send WhatsApp Messages
- **As a user, I can** click the WhatsApp tab to message the client

### Create Case
- **As a user, I can** click "Create Case" to start a mortgage application
- **As a user, I can** select mortgage type (Islamic/Conventional)
- **As a user, I can** select banks to apply with
- **As a user, I can** add handover notes

### View Client's Cases
- **As a user, I can** see all cases for this client
- **As a user, I can** click on a case to view its details

### Mark as Not Proceeding
- **As a user, I can** mark a client as "Not Proceeding" if they decide not to continue
- **As a user, I can** add a reason

### Mark as Not Eligible
- **As a user, I can** mark a client as "Not Eligible" if they don't qualify
- **As a user, I can** add a reason

### View Activity History
- **As a user, I can** click "Activity" tab to see all activity
- **As a user, I can** see calls, notes, document uploads, and status changes

---

## Client Statuses

| Status | Meaning |
|--------|---------|
| **Active** | Currently working with this client |
| **Not Proceeding** | Client decided not to continue |
| **Not Eligible** | Client doesn't qualify for a mortgage |

---

## Eligibility Status

| Status | Meaning |
|--------|---------|
| **Eligible** | DBR and LTV are within limits |
| **Not Eligible** | DBR or LTV exceeds limits |
| **Pending** | Missing financial info to calculate |

---

## Document Types

| Document | What it is |
|----------|------------|
| Emirates ID | UAE national ID card |
| Salary Certificate | Proof of employment and salary |
| Bank Statements | 3-6 months of bank history |
| Other | Any additional supporting documents |

---

## Quick Actions (Table Row)

| Icon | Action |
|------|--------|
| Note icon | Add a quick note |
| Phone icon | Log a call |
| Case chips | Click to view a case |
| "..." menu | Create Case, Mark Not Proceeding, Mark Not Eligible |

---

## Tips
1. Upload all required documents before creating a case
2. Make sure financial info is complete for accurate eligibility
3. Add detailed notes before creating a case for context
4. Keep documents organized - each type has its own slot
